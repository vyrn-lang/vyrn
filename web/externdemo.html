Vela × JS interop

examples/externdemo.vela declares three extern fn imports (RFC-0012) and calls them. The host page below supplies them in the vela import namespace: jsLog (a String decoded from linear memory), jsNow (returns a Float64), and jsAdd (two Int64s round- tripped through JS). The same module traps under the interpreter and the native binary — only wasm has a host. ← plain demo
